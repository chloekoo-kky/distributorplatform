Upload Products

Step of 2

Select a .csv or .xlsx file to import. Use the same columns as Export products to Excel. Rows are matched by sku for updates. The exported id column is ignored on import.

Updating: only sku is required to find the product. If name, categories, suppliers, or other optional columns are left blank, existing values for that product are kept.

New products: if sku is blank, one is generated from the English name. If name is blank but sku is set, the name defaults to that SKU.

Columns include sku, name, display_order, description, origin_country, members_only (1/0), categories, suppliers, selling_price, profit_margin, and read-only base_cost (export only).
